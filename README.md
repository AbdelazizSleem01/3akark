# 3akark - Real Estate Web App 🏠

**3akark** is a modern real estate web application built with **Next.js**, **Tailwind CSS**, **DaisyUI**, and **MongoDB**.  
It allows users to browse, add, and manage real estate listings with a sleek and responsive interface.

---

## 🔧 Tech Stack

- **Frontend**: Next.js 14, Tailwind CSS, DaisyUI  
- **Backend**: Node.js, MongoDB  
- **Authentication**: JWT  
- **Features**: CRUD system, notifications, property filtering, image upload, admin & moderator roles

---

## ✨ Key Features

- 🏘️ Create, edit, and delete property listings  
- 📷 Upload multiple property images  
- 🔍 Filter properties by category, type, or location  
- 👤 User authentication and profile editing  
- 🔔 Real-time notifications for admins and moderators  
- 📝 Rich text editor for property descriptions  
- 📄 Generate and print PDF invoices for bookings  
- 🌐 Arabic language support
